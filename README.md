# Iraqi Currency Authenticity Detector

This project implements a computer vision system to verify the authenticity of currency notes using feature matching techniques.

## Methodology
The system utilizes the ORB algorithm to extract keypoints from a reference bank note and compare them with a test image for validation.

- Language: Python 3.x
- Library: OpenCV
- Algorithm: Feature detection and Brute-Force matching

## Files
- main.py: Core authentication logic
- original_note.jpg.JPG: Reference template
- test_note.jpg.jpg: Input image for validation
